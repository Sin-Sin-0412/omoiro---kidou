# 思色軌道 (Omoiro Kido)

都電7500方に乗っているEvil Neuroを視覚化した3Dサイト。

## 概要
夕暮れ時の静寂の中を走る電車とEvilの姿を、Three.jsを用いた3Dレンダリングとポストプロセッシングで表現しています。

## 公開URL
[https://omoiro-kidou.netlify.app/](https://omoiro-kidou.netlify.app/)

## 使用技術
- **HTML5 / CSS3** (Responsive Design, dvh units)
- **JavaScript** (ES6+)
- **Three.js** (WebGL, GLTFLoader, Post-processing)
- **GitHub & Netlify** (CI/CD)

## 制作のこだわり
- 100dvh を活用したモバイルフレンドリーなレイアウト。
- 夕焼けの光を反射するモデルと、空気感を演出する霧（Fog）や埃（Dust）のパーティクル。
- Evilの視線や呼吸、記憶の中の情景を感じさせるアニメーション・演出。
